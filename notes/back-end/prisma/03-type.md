# type

| 类型      | 描述                             |
|-----------|----------------------------------|
| `String`  | 表示文本字符串                   |
| `Int`     | 表示整数                         |
| `Float`   | 表示浮点数（小数）               |
| `Boolean` | 表示布尔值（true 或 false）       |
| `DateTime`| 表示日期和时间                   |
| `Json`    | 表示 JSON 对象或数组             |
| `Bytes`   | 表示字节序列，通常用于存储二进制数据 |
| `Decimal` | 精确表示十进制数字（注意：仅在某些数据库如 PostgreSQL 中完全支持） |
